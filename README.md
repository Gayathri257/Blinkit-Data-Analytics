# 🛒 Blinkit Data Analytics Project

## 📌 Overview

This project focuses on optimizing Blinkit's end-to-end business operations using comprehensive data analytics and predictive modeling techniques. Blinkit is a quick-commerce grocery delivery platform aiming to enhance efficiency, improve customer engagement, and drive profitable growth through data-driven insights.

---

## 📊 Dataset Description

The dataset includes:
- **Customer data**: Demographics, order history, feedback, segments
- **Order & Delivery data**: Time, partner performance, delays
- **Product & Inventory**: Stock levels, damaged stock, reorder points
- **Marketing campaigns**: Spend, impressions, clicks, conversions
- **Customer feedback**: Sentiment scores, review text

---

## 🧠 Project Objectives

- Analyze customer behavior and segmentation  
- Evaluate delivery performance and delay reasons  
- Monitor product and inventory trends  
- Measure marketing campaign effectiveness  
- Perform sentiment analysis on customer feedback  
- Build predictive models for delivery time, segmentation, and sales forecasting

## 🐍 Part 3: Python Analytics & ML Models

### 🔎 Exploratory Data Analysis (EDA)
- Cleaned and prepared datasets  
- Stats for delivery time, order value  
- Distribution plots and segment comparisons  

### 💬 Sentiment Analysis
- NLP techniques for feedback text  
- Correlation between sentiment and delivery/order value  

### ⏱️ Delivery Time Prediction (Regression)
- Used factors: distance, delivery partner, promised time  
- Evaluated using RMSE, MAE  

### 👥 Customer Segmentation (Clustering)
- Features: Order frequency, avg. value, segment  
- KMeans clustering for behavioral groups
